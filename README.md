Gestor de Tarefas 📝
Projeto simples de API REST desenvolvido com o objetivo de praticar conceitos de backend utilizando Spring Boot.

📚 Sobre o projeto
Este projeto foi criado durante meus estudos em desenvolvimento backend para entender melhor como funciona a construção de uma API REST, organização em camadas e persistência de dados em memória.

A aplicação permite realizar operações básicas de cadastro e gerenciamento de usuários e tarefas.

🚀 Tecnologias utilizadas
Java 25
Spring Boot
Spring Web
Spring Data JPA
Banco em memória H2
Maven
⚙️ Funcionalidades
Cadastro de usuários
Criação de tarefas
Listagem de dados
Relacionamento entre entidades
▶️ Como executar
Clonar o repositório
Abrir o projeto na IDE
Executar a classe principal
Acessar o console H2 em: http://localhost:8080/h2-console
Projeto desenvolvido para fins de estudo 📖
